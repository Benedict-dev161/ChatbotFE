# Disty Teknologi Chatbot Dashboard

Project Laravel untuk dashboard monitoring chatbot customer service.

## Fitur

- Dashboard statistik customer
- Monthly customers chart
- Unanswered customers
- Total percakapan
- Eskalasi ke CS
- Average response time
- Aktivitas terbaru
- Chat management
- Staff page

## Tech Stack

- Laravel
- MySQL / MariaDB
- Blade
- CSS
- PHP
